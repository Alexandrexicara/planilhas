Coloque aqui o instalador desktop (.exe) para liberar o download no site.

Exemplo:
- SistemaPlus_v2.0.0.exe

Regras:
1. O sistema usa automaticamente o .exe mais recente desta pasta.
2. Rota de download: /download-exe
3. A tela sistema.html mostra:
   - nome do arquivo publicado
   - data de atualizacao
   - limite de maquinas por licenca
